# Rust Learning Mandates

This project is dedicated to mastering Rust from the perspective of a senior full-stack engineer.

## 🎯 Learning Focus
- **Deep Understanding:** Don't just make it compile; understand *why* the borrow checker is complaining.
- **Idiomatic Rust:** Prioritize `clippy` suggestions and idiomatic patterns (e.g., using `map`, `unwrap_or`, pattern matching).
- **Surgical Edits:** Use the `replace` tool for code changes to maintain clarity.

## 🛠 Tools & Commands
- **Check:** `cargo check` (fast verification)
- **Lint:** `cargo clippy` (essential for learning idioms)
- **Format:** `cargo fmt`
- **Test:** `cargo test`

## 📖 Progress Tracking
- Use `README.md` as the primary roadmap.
- Create new modules/files in `src/` for each major concept (e.g., `src/ownership.rs`, `src/traits.rs`).
